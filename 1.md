````md
```json

  "relationships": [],

}
````

### 5.2 Field Definitions

| Field         | Description                               |

| id            | Unique identifier for the entity          |
| type          | Entity type (e.g., class, method, config) |
| attributes    | Key-value metadata                        |
| relationships | Links to other entities                   |
| confidence    | 0–1 score indicating reliability          |
| source        | Origin of data                            |
| timestamp     | Processing time                           |

### 5.3 Output Rules

* Output MUST be valid JSON
* No null fields unless explicitly allowed
* Confidence scoring is mandatory
* Every relationship must reference a valid entity
